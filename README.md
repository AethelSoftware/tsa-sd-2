


# TRYVER
- This is our Software Development project for the 2025-2026 Technology Student Association Pennsylvania State competition.
- Developed by Steven Hu, Kanisk Gaurav Prakash, and Joshua Shah

## Primary Purpose of our Application 
- Drawing inspiration from existing navigation tools and seeking to rectify their flaws, the purpose of our application is to create such an application that increases accessibility for impaired users. 

## Instructions on Using this Application
1. Clone the Repository 

git clone <https://github.com/AethelSoftware/tsa-sd-2.git>
cd tsa-sd-2/backend

2. Ensure that Python Version **3.10** is installed (or that you are using a venv with this version)

- Download Python 3.10
- python3.10 -m venv venv
- source venv/bin/activate   # macOS/Linux
- venv\Scripts\activate      # Windows

3. Install Dependencies

- pip install --upgrade pip
- pip install -r requirements.txt

Should installation fail, please run this command (or edit the config)

- pip install -r requirements.txt --index-url https://pypi.org/simple

4. Run the backend 

- python app.py
**NOTE:** Do not run as a module.

5. Run the frontend 

- cd ../frontend
- npm install
- npm run dev
